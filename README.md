#1. Para o programa funcionar, é necessário uma API particular da open AI, da qual precisa ser inserida dentro do arquivo .env.

#2. Execute o arquivo ingestion.py para criar o banco de dados vetorizado e, em seguida, o arquivo app_chat.py para testar no terminal. 

#3. Por fim, para usar a "simulação de aplicativo", execute "python -m streamlit run app_gui.py", sem aspas, para ter acesso ao projeto contendo o front-end.
